# Posts Node Structure - Dashboard.jsx

```
├─ Posts
│   ├─ <postId1>
│   │    ├─ postId: "string"
│   │    ├─ message: "string"
│   │    ├─ postUrl: "url" | null
│   │    ├─ adminId: "string"
│   │    ├─ adminName: "string"
│   │    ├─ adminProfile: "url"
│   │    ├─ time: "ISO_string"
│   │    ├─ likeCount: number
│   │    ├─ likes
│   │    │    ├─ <userId1>: true | false
│   │    │    ├─ <userId2>: true | false
│   │    │    └─ <adminId>: true | false
│   │    └─ seenBy
│   │         ├─ <userId1>: true | false
│   │         ├─ <userId2>: true | false
│   │         └─ <adminId>: true | false
│   ├─ <postId2>
│   └─ ...
│
├─ Users
│   ├─ <userId1>
│   │    ├─ userId: "string"
│   │    ├─ username: "string"
│   │    ├─ name: "string"
│   │    ├─ profileImage: "url"
│   │    └─ role: "admin" | "student" | "teacher"
│   ├─ <userId2>
│   └─ ...
│
├─ Teachers
│   ├─ <teacherId1>
│   │    ├─ userId: "string"
│   │    ├─ name: "string"
│   │    └─ profileImage: "url"
│   ├─ <teacherId2>
│   └─ ...
│
├─ Students
│   ├─ <studentId1>
│   │    ├─ userId: "string"
│   │    ├─ name: "string"
│   │    ├─ profileImage: "url"
│   │    └─ grade: "string"
│   ├─ <studentId2>
│   └─ ...
│
├─ Parents
│   ├─ <parentId1>
│   │    ├─ userId: "string"
│   │    ├─ name: "string"
│   │    ├─ profileImage: "url"
│   │    └─ studentId: "string"
│   ├─ <parentId2>
│   └─ ...
│
└─ Chats
    ├─ <adminId_userId>
    │    └─ messages
    │         ├─ <messageId1>
    │         │    ├─ senderId: "string"
    │         │    ├─ receiverId: "string"
    │         │    ├─ message: "string"
    │         │    ├─ time: "ISO_string"
    │         │    └─ seen: true | false
    │         └─ <messageId2>
    └─ <userId_adminId>
         └─ messages
              ├─ <messageId1>
              └─ <messageId2>
```

## Field Details

### Posts Node Fields
- **postId**: Unique identifier generated by backend
- **message**: Post text content from `postText` state
- **postUrl**: Media URL from `postMedia` upload (null if no media)
- **adminId**: Admin user ID who created the post (`admin.userId`)
- **adminName**: Admin display name (`admin.name`)
- **adminProfile**: Admin profile image URL (`admin.profileImage`)
- **time**: ISO timestamp when post was created
- **likeCount**: Total number of likes (calculated from likes object)
- **likes**: Object with user IDs as keys and boolean values
- **seenBy**: Object tracking which users have seen the post

### Users Node Fields
- **userId**: Firebase unique key
- **username**: User's username
- **name**: User's display name
- **profileImage**: Profile picture URL
- **role**: User role ("admin", "student", "teacher")

### Teachers Node Fields
- **userId**: Reference to Users node
- **name**: Teacher's display name
- **profileImage**: Teacher's profile image

### Students Node Fields
- **userId**: Reference to Users node
- **name**: Student's display name
- **profileImage**: Student's profile image
- **grade**: Student's grade/class

### Parents Node Fields
- **userId**: Reference to Users node
- **name**: Parent's display name
- **profileImage**: Parent's profile image
- **studentId**: Reference to student's userId

### Chats Node Fields
- **messages**: Array of message objects
- **senderId**: ID of message sender
- **receiverId**: ID of message receiver
- **message**: Message text content
- **time**: ISO timestamp
- **seen**: Boolean indicating if message was read
